Accessibility

Date & time picker supports a number of keyboard shortcuts.

Open / close the picker

  • Alt + Arrow Down - open the picker
  • Esc, Alt + Arrow Up - close the picker

Navigating the month view

  • Arrow Left - previous day
  • Arrow Right - next day
  • Arrow Up - same day in the previous week
  • Arrow Down - same day in the next week
  • Home - first day of the month
  • End - last day of the month
  • Page Up - same date in the previous month
  • Page Down - same date in the next month
  • Alt + Page Up - same date in the previous year
  • Alt + Page Down - same date in the next year
  • Enter - select the highlighted date

Navigating the year view

  • Arrow Left - previous month
  • Arrow Right - next month
  • Arrow Up - move 3 months back
  • Arrow Down - move 3 months forward
  • Home - first month of the year
  • End - last month of the year
  • Page Up - same month in the previous year
  • Page Down - same month in the next year
  • Alt + Page Up - move 10 years back
  • Alt + Page Down - move 10 years forward
  • Enter - select the highlighted month

Navigating the multi-year view

  • Arrow Left - previous year
  • Arrow Right - next year
  • Arrow Up - move 3 years back
  • Arrow Down - move 3 years forward
  • Home - first year of the current range
  • End - last year of the current range
  • Page Up - previous range (move 21 years back)
  • Page Down - next range (move 21 years forward)
  • Alt + Page Up - move 210 years back
  • Alt + Page Down - move 210 years forward
  • Enter - select the highlighted year
