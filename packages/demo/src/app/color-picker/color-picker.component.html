Angular Color Picker Directive

A Color Picker Directive for Angular with no dependencies.

based on angular2-color-picker and updates from ngx-color-picker

Usage:

<input [(cpColor)]="color"
       [style.background]="color"/>

Or:

<input [style.background]="color"
       [cpColor]="color"
       (cpColorChange)="color=$event"/>

Grayscale color mode:

<input [(cpColor)]="color" 
       [cpMode]="'grayscale'"
       [style.background]="color"/>

Show the color in the input field:

<input [value]="color"
       [style.background]="color"
       [(cpColor)]="color"/>


Output format:

<input [value]="color"
       [style.background]="color"
       [cpOutputFormat]="'rgba'"
       [(cpColor)]="color"/>

Changing dialog position:

<input [value]="color"
       [style.background]="color"
       [cpPosition]="'bottom'"
       [(cpColor)]="color"/>
Change me!

You can introduce a offset of the color picker relative to the html element:

<span [style.color]="color"
      [cpPosition]="'bottom'"
      [cpPositionOffset]="0.5"
      [(cpColor)]="color">Change me!</span>

Show cancel button:

<input [value]="color"
       [style.background]="color"
       [cpCancelButton]="true"
       [(cpColor)]="color"/>

Change cancel button class, in this example we are using a bootstrap button:

<input [value]="color"
       [style.background]="color"
       [cpCancelButton]="true"
       [(cpColor)]="color"/>

Show OK button:

<input [value]="color"
       [style.background]="color"
       [cpOKButton]="true"
       [cpSaveClickOutside]="false"
       [(cpColor)]="color"/>

With preset colors:

<input [style.background]="color"
       [cpPresetColors]="['#fff', '#000', '#2889e9', '#e920e9', '#fff500', 'rgb(236,64,64)']"
       [(cpColor)]="color"/>

Add and remove preset colors:

<input [style.background]="color"
       [cpAlphaChannel]="'always'"
       [cpOutputFormat]="'rgba'"
       [cpPresetColors]="['#fff', '#2889e9']"
       [cpAddColorButton]="true"
       [(cpColor)]="color"/>


Toggle status: {{toggle}}

Use cpToggle with cpIgnoredElements:

<input #ignoredInput
       [style.background]="color"
       [cpIgnoredElements]="[ignoredButton, ignoredInput]"
       [(cpToggle)]="toggle"
       [(cpColor)]="color"/>
<button #ignoredButton (click)="toggle=!toggle"></button>



Change alpha channel behaviour:

<input [value]="color"
       [style.background]="color"
       [cpAlphaChannel]="'always'"
       [cpOutputFormat]="'rgba'"
       [(cpColor)]="color"/>
<input [value]="color"
       [style.background]="color13"
       [cpAlphaChannel]="'disabled'"
       [cpOutputFormat]="'rgba'"
       [(cpColor)]="color"/>
<input [value]="color"
       [style.background]="rgbaText"
       [cpAlphaChannel]="'always'"
       [cpOutputFormat]="'hex'"
       [cpColor]="color"
       (cpColorChange)="rgbaText=onChangeColorHex8($event);color16=$event"/>
<input [value]="color"
       [style.background]="color"
       [cpAlphaChannel]="'forced'"
       [cpOutputFormat]="'hex'"
       [(cpColor)]="color"/>

Show the dialog permanently: 

<span [style.background]="arrayColors[selectedColor]"
      [cpToggle]="true"
      [cpDialogDisplay]="'inline'"
      [cpCancelButton]="true"
      [(cpColor)]="arrayColors[selectedColor]"></span>
<div [style.background]="arrayColors['color1']"
     (click)="selectedColor='color1'"></div>
<div [style.background]="arrayColors['color2']"
     (click)="selectedColor='color2'"></div>

Options Values (default values in bold)
cpColor The color to show in the color picker dialog.
cpWidth '230px'
Use this option to set color picker dialog width (pixels).
cpHeight 'auto'
Use this option to force color picker dialog height (pixels).
cpToggle false, true
Input/ouput to open/close the color picker.
cpDisabled false, true
Disable colorpicker.
cpMode 'color', 'grayscale', 'presets'
Dialog color mode.
cpCmykEnabled false, true
Enable cmyk color mode.
cpOutputFormat OutputFormat.auto
OutputFormat.hex
OutputFormat.rgba
OutputFormat.hsla
cpAlphaChannel AlphaChannel.enabled
AlphaChannel.enabled: alpha channel is not included for hexadecimal (hex6) values or for values without alpha (alpha = 1).
AlphaChannel.disabled: alpha channel is completely disabled.
AlphaChannel.always: alpha channel is included for hexadecimal (hex6) values and values without alpha (alpha = 1).
AlphaChannel.forced: alpha channel field is added for hexadecimal (hex6) values.
cpFallbackColor '#fff'
Is used when the color is not well-formed or not defined.
cpPosition DialogPosition.right
DialogPosition.left
DialogPosition.top
DialogPosition.bottom
cpPositionOffset 0
Dialog offset (percent) relative to the element that contains the directive.
cpPresetLabel true
Show label for preset colors. If string is given, it overrides the default label.
cpPresetColors []
Array of preset colors to show in the color picker dialog.
cpDisableInput false, true
Disables / hides the color input field from the dialog.
cpDialogDisplay DialogDisplay.popup
DialogDisplay.popup: dialog is showed when user clicks in the directive.
DialogDisplay.inline: dialog is showed permanently. You can show/hide the dialog with cpToggle.
cpIgnoredElements []
Array of HTML elements that will be ignored by the color picker when they are clicked.
cpSaveClickOutside true, false
If true the initial color is restored when user clicks outside.
cpCloseClickOutside true, false
If true the dialog closes when user clicks outside.
cpOKButton false, true
Shows the Ok button. Saves the selected color.
cpCancelButton false, true
Shows the Cancel button. Cancel the selected color.
cpAddColorButton false, true
Add or remove colors into your preset panel. The [cpPresetColors] is needed
cpMaxPresetColors 6 (number)
Use this option to set the max colors allowed into preset panel.
cpUseRootViewContainer false, true
Create dialog component in the root view container instead the elements view container.

Events Description (data format in bold)
cpOpen Current color value, send when dialog is opened. (value: string)
cpClose Current color value, send when dialog is closed. (value: string)
cpInputChange Input name and its value, send when user changes color through inputs. (value: InputChangeEvent)
cpToggleChange Status of the dialog, send when dialog is opened / closed. (open: boolean)
cpSliderDragStart Slider name and current color, send when slider dragging starts. (value: SliderChangeEvent)
cpSliderChange Slider name and its value, send when user changes color through slider. (value: SliderChangeEvent)
cpSliderDragEnd Slider name and current color, send when slider dragging ends. (value: SliderChangeEvent)
cpColorSelectCancel Color select canceled, send when Cancel button is pressed. (void)
cpColorSelect Selected color value, send when user presses the OK button or clicks outside (if cpSaveClickOutside is true). (value: string)
cpColorChange Changed color value, send when color is changed. (value: string)
cpCmykColorChange Changed CMYK value, send when CMYK is enabled. (value: string)
cpPresetColorsChange Preset colors value, send when Add Color button is pressed. (value: array)
