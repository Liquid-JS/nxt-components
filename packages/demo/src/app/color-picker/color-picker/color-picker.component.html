Angular Color Picker Directive

A Color Picker Directive for Angular with no dependencies.

based on angular2-color-picker and updates from ngx-color-picker

  • {{ example.name }}
Options Values (default values in bold)
nxtColor The color to show in the color picker dialog.
width '230px'
Use this option to set color picker dialog width (pixels).
height 'auto'
Use this option to force color picker dialog height (pixels).
toggle false, true
Input/ouput to open/close the color picker.
disabled false, true
Disable colorpicker.
mode 'color', 'grayscale', 'presets'
Dialog color mode.
cmykEnabled false, true
Enable cmyk color mode.
outputFormat OutputFormatEnum.auto
OutputFormatEnum.hex
OutputFormatEnum.rgba
OutputFormatEnum.hsla
alphaChannel AlphaChannelEnum.enabled
AlphaChannelEnum.enabled: alpha channel is not included for hexadecimal (hex6) values or for values without alpha (alpha = 1).
AlphaChannelEnum.disabled: alpha channel is completely disabled.
AlphaChannelEnum.always: alpha channel is included for hexadecimal (hex6) values and values without alpha (alpha = 1).
AlphaChannelEnum.forced: alpha channel field is added for hexadecimal (hex6) values.
fallbackColor '#fff'
Is used when the color is not well-formed or not defined.
position DialogPositionEnum.right
DialogPositionEnum.left
DialogPositionEnum.top
DialogPositionEnum.bottom
positionOffset 0
Dialog offset (percent) relative to the element that contains the directive.
presetLabel true
Show label for preset colors. If string is given, it overrides the default label.
presetColors []
Array of preset colors to show in the color picker dialog.
disableInput false, true
Disables / hides the color input field from the dialog.
dialogDisplay DialogDisplayEnum.popup
DialogDisplayEnum.popup: dialog is showed when user clicks in the directive.
DialogDisplayEnum.inline: dialog is showed permanently. You can show/hide the dialog with toggle.
ignoredElements []
Array of HTML elements that will be ignored by the color picker when they are clicked.
saveClickOutside true, false
If true the initial color is restored when user clicks outside.
closeClickOutside true, false
If true the dialog closes when user clicks outside.
okButton false, true
Shows the Ok button. Saves the selected color.
cancelButton false, true
Shows the Cancel button. Cancel the selected color.
presetColorsEditable false, true
Add or remove colors into your preset panel. The [presetColors] is needed
maxPresetColors 6 (number)
Use this option to set the max colors allowed into preset panel.
useRootViewContainer false, true
Create dialog component in the root view container instead the elements view container.

Events Description (data format in bold)
open Current color value, send when dialog is opened. (value: string)
close Current color value, send when dialog is closed. (value: string)
inputChange Input name and its value, send when user changes color through inputs. (value: InputChangeEvent)
toggleChange Status of the dialog, send when dialog is opened / closed. (open: boolean)
sliderDragStart Slider name and current color, send when slider dragging starts. (value: SliderChangeEvent)
sliderChange Slider name and its value, send when user changes color through slider. (value: SliderChangeEvent)
sliderDragEnd Slider name and current color, send when slider dragging ends. (value: SliderChangeEvent)
colorSelectCancel Color select canceled, send when Cancel button is pressed. (void)
colorSelect Selected color value, send when user presses the OK button or clicks outside (if saveClickOutside is true). (value: string)
nxtColorChange Changed color value, send when color is changed. (value: string)
cmykColorChange Changed CMYK value, send when CMYK is enabled. (value: string)
presetColorsChange Preset colors value, send when Add Color button is pressed. (value: array)
